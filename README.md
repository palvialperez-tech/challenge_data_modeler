# challenge_data_modeler
Analizador de campaña de “3 cuotas sin interés” de forma autónoma y confiable. Actualmente existen datos crudos de clientes, tarjetas, transacciones, comercios y eventos de campaña. El problema es que las métricas no están estandarizadas, hay duplicidad de lógica y los equipos de negocio dependen demasiado de consultas ad hoc..
